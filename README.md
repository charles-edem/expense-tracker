# Expense Tracker App
A simple and interactive Expense Tracker built with Vanilla JavaScript.  
It allows users to track income and expenses, manage budgets, and monitor financial balance in real time using local storage.


# Features
1. Add income transactions
2. Add expense transactions
3. Delete transactions
4. Track total income, expenses, and balance
5. Set a monthly budget
6. Visual budget progress bar (warning colors when close to limit)
7. Filter transactions by:
  - Income
  - Expenses
  - Category
8. Data persistence using Local Storage (data stays after refresh)



# Built With
1 HTML
2. CSS
3. JavaScript (ES6 Modules)



# Project Structure
main.js : App logic and event handling
ui.js : DOM rendering and UI updates
storage.js : Local storage handling
validation.js : Input validation rules
